# results/
This folder contains exported intermediate result tables supporting the main figures and project report.

## `results_screening_vKO_screening_summary.csv`
Summary of multi-gene virtual perturbation screening, including mean distance, maximum distance, and number of significant DR genes.

## `results_figure1/`
Node and edge tables for Figure 1A RAC1-centered GRN rewiring network and Figure 1B WT coordination network.

## `results_figure2/`
Tables supporting the perturbation-informed latent manifold analysis, including automatic/refined state annotations, cluster-level UCell program scores, and UMAP coordinates with program scores and functional gradients.

## `results_figure3/`
Tables supporting TCGA-LUAD prognostic modeling, including LASSO-selected genes, patient risk scores, Cox regression results, time-dependent ROC AUCs, and C-index results.
