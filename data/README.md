# data/

This folder is intended for input data files or small metadata files.

Large raw matrices, `.rds`, `.RData`, and GEO/TCGA raw data should generally not be uploaded to GitHub.
Instead, document the download source and expected filenames here.

Expected local files include:
- `GSE131907_annotation.txt.gz`
- `GSE131907_UMI_matrix.rds`
- `RAC1_vko.rds`
- `RAC1_TopDRgenes.csv`
