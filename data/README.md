# data/
This directory contains the core input data files required to run the downstream analysis scripts. All files are generated by the preprocessing and virtual knockout pipeline (`R/00_preprocessing_and_vKO.R`).

## file description
- `expr_sub.rds`  
  Quality-controlled, filtered, and down-sampled single-cell expression matrix (Seurat object) containing 5,000 cells.
- `RAC1_TopDRgenes.csv`
  Top 100 differentially rewired (DR) genes identified after virtual RAC1 knockout, ranked by network distance.
- `WT_GRN_matrix.rds`
  Gene regulatory network adjacency matrix under wild-type conditions (from scTenifoldKnk tensor networks).
- `RAC1_vKO_GRN_matrix.rds`  
  Gene regulatory network adjacency matrix after virtual RAC1 knockout.
- `RAC1_vko.rds`
  Complete scTenifoldKnk output object containing tensor networks (`WT` and `KO`) and differential regulation results (`diffRegulation`).

## raw data source
- Dataset: GSE131907 (LUAD tumor epithelial cells)
- Access: NCBI Gene Expression Omnibus (GEO), https://www.ncbi.nlm.nih.gov/geo/query/acc.cgi?acc=GSE131907
- Note: The original UMI count matrix (`GSE131907_UMI_matrix.rds`) and cell annotation file (`GSE131907_annotation.txt.gz`) are large and not included in this repository. Please download them from GEO and place them in this directory, or update the file paths in `R/0_preprocessing_and_vKO.R` accordingly.

## Generation Pipeline
All files in this directory are automatically generated by running the `R/00_preprocessing_and_vKO.R` script. Ensure the raw data files are in place before executing the script.
