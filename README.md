# RAC1-latent-state-mapping
Perturbation-informed latent tumor-state mapping reveals RAC1-responsive regulatory network rewiring in LUAD.

# Quick Start
Run the following scripts in order:
1. `R/00_preprocessing_and_vKO.R` – Data preprocessing, virtual knockout, select RAC1
2. `R/Figure1_GRN_rewiring.R` – GRN rewiring network (Fig.1A) and WT coordination network (Fig.1B)
3. `R/Figure2_latent_manifold.R` – Perturbation-informed manifold and functional program analysis (Fig.2A–D)
4. `R/Figure3_prognostic_model.R` – TCGA prognostic model and validation (Fig.3A–C)
For complete reproduction, run `R/complete_analysis_pipeline.R`.
