# R scripts

This folder contains the main R analysis scripts for the RAC1 perturbation project.

## Script overview

- `00_preprocessing_and_vKO.R`  
  Single-cell preprocessing, tumor epithelial cell extraction, candidate gene prioritization, and scTenifoldKnk virtual knockout screening.

- `Figure1_GRN_rewiring.R`  
  RAC1-centered GRN rewiring analysis and WT coordination network visualization.

- `Figure2_latent_manifold.R`  
  Perturbation-informed latent manifold construction, UCell program scoring, state annotation, and functional gradient analysis.

- `Figure3_prognostic_model.R`  
  TCGA-LUAD prognostic modeling using RAC1-responsive genes.

- `complete_analysis_pipeline.R`  
  full script for running the complete analysis workflow.
