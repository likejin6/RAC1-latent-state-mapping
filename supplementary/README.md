# supplementary/

This folder contains supplementary tables supporting the RAC1 perturbation project report.

## Files

- Supplementary_Table_S1_vKO_screening_summary.csv  
  Summary of multi-gene virtual perturbation screening.

- Supplementary_Table_S2_RAC1_TopDRgenes.csv  
  Top RAC1-responsive differentially rewired genes ranked by perturbation distance.

- Supplementary_Table_S3_cluster_program_summary.csv  
  Cluster-level UCell program scores and hybrid state annotations.

- Supplementary_Table_S4_prognostic_model_summary/  
  Summary files for TCGA-LUAD prognostic modeling.

- Supplementary_Table_S5_program_gene_sets.csv  
  Marker-based functional program gene sets used for UCell scoring.

Future extension may include Python-based network topology validation and node2vec graph embedding analysis.
